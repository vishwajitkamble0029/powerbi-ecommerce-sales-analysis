# 📊 Madhav E-Commerce Sales Dashboard

## Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales performance and generate actionable business insights.

The dashboard helps monitor sales trends, profitability, customer behavior, and regional performance using interactive visualizations and KPI reporting.

---

## Objectives

- Analyze overall sales performance
- Monitor profit and revenue trends
- Identify high-performing products and customers
- Compare regional sales performance
- Support data-driven business decisions

---

## Tools Used

- Power BI
- Microsoft Excel
- Data Visualization
- Business Analytics

---

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value (AOV)

---

## Dashboard Features

- Monthly Profit Analysis
- State-wise Sales Analysis
- Customer Performance
- Product Category Analysis
- Payment Mode Analysis
- Interactive Filters

---

## Business Insights

- Identified profitable states and customer segments.
- Compared monthly profit trends to identify seasonal performance.
- Evaluated product category contribution to overall sales.
- Analyzed customer purchasing patterns for better business decisions.
